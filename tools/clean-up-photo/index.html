Questions, answered

Is this clean up photo tool free?

Yes. You can clean up photo online, preview the result and download a PNG without sign-up.

Are photos uploaded to a server?

No. The cleanup runs in your browser with Canvas. GitHub Pages is static and no backend receives your images.

What does cleanup do?

It estimates a small straighten angle from image edges, scales the rotated image to avoid black corners, blends light denoise, adds unsharp-mask clarity and lightens dark vignette corners.

Can it remove objects from photos?

This web tool focuses on tidy, clear photo cleanup rather than object removal. For a better mobile cleanup workflow, try PicClear.
